# 2.5 JDBC / REST & Jobs

## Exam objective

> Use JDBC/ODBC or REST clients in notebooks to land data into cloud storage or directly into Unity-Catalog-governed tables, usually orchestrated and scheduled with Lakeflow Jobs.

## Must know

- **JDBC**: `spark.read.format("jdbc")` with url, dbtable or query, credentials.
- **Secrets**: `dbutils.secrets.get(scope, key)` — never hardcode passwords.
- **REST**: `requests` / `http` in notebook → DataFrame → Delta; mind driver memory on large JSON.
- **Land to storage** vs **UC table**: staging path then Auto Loader vs direct `saveAsTable`.
- **Lakeflow Jobs**: schedule notebook task with job cluster; parameters for incremental windows.

## When to use what

| Approach | When |
|----------|------|
| Direct JDBC → UC Delta | Small/medium tables, simple schedule |
| JDBC → cloud files → Auto Loader | Large extracts, reuse file pipeline |
| REST one-shot | APIs without Connect connector |

## PySpark / API essentials

```python
user = dbutils.secrets.get("prod", "db_user")
password = dbutils.secrets.get("prod", "db_password")

df = (spark.read.format("jdbc")
  .option("url", "jdbc:postgresql://host:5432/db")
  .option("dbtable", "public.orders")
  .option("user", user)
  .option("password", password)
  .option("numPartitions", 8)
  .option("partitionColumn", "order_id")
  .option("lowerBound", 1)
  .option("upperBound", 1000000)
  .load())

df.write.format("delta").mode("append").saveAsTable("main.bronze.pg_orders")

# REST pattern (small payload)
import requests, pandas as pd
data = requests.get(api_url, headers={"Authorization": f"Bearer {token}"}).json()
pdf = pd.DataFrame(data["items"])
spark.createDataFrame(pdf).write.mode("append").saveAsTable("main.bronze.api_events")
```

## UI / product surfaces

- **Secret scopes**; **Jobs** → notebook task + schedule; **Partner Connect** for some JDBC sources.

## Exam traps

- **Driver OOM** collecting huge REST response to driver.
- JDBC without **partitionColumn** on large table (single-thread read).
- No **Job** schedule for production notebook-only manual runs.

- [ ] Read a small JDBC table into bronze UC table using secrets.
