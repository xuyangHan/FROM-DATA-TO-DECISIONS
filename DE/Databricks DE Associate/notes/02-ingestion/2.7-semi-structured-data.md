# 2.7 Semi-structured & unstructured data

## Exam objective

> Ingest semi-structured and unstructured data (for example, JSON and nested data) via Lakeflow Connect and other managed connectors into Unity-Catalog-governed Delta tables.

## Must know

- **JSON**: `read.format("json")` or Auto Loader `cloudFiles.format=json`; nested structs and arrays.
- **from_json** / **schema_of_json** — parse string column to struct.
- **explode** — flatten arrays after landing (see 3.3).
- **Variant** type (where supported) for flexible JSON in Delta.
- Connect can land semi-structured SaaS objects into bronze; refine in silver.
- Schema inference risky on sparse JSON — provide schema or schemaHints.

## When to use what

| Data | Ingest | Transform |
|------|--------|-----------|
| JSON files in cloud | Auto Loader + schemaEvolution | Silver: cast, explode |
| Nested API via Connect | Connect → bronze | PySpark flatten |
| Raw text/images | Volume + metadata table | Downstream ML (awareness) |

## PySpark / API essentials

```python
# JSON with explicit schema
from pyspark.sql.types import StructType, StructField, StringType, ArrayType

schema = StructType([
  StructField("id", StringType()),
  StructField("tags", ArrayType(StringType())),
  StructField("attrs", StructType([
    StructField("color", StringType()),
  ])),
])

raw = spark.read.schema(schema).json("/Volumes/main/bronze/inbox/events/")

silver = (raw
  .withColumn("tag", explode("tags"))
  .select("id", "tag", "attrs.color")
  .filter("id IS NOT NULL"))

silver.write.format("delta").mode("overwrite").saveAsTable("main.silver.events")

# Parse JSON string column
from pyspark.sql.functions import from_json, schema_of_json
sample = spark.table("main.bronze.logs").select("payload").limit(100)
json_schema = schema_of_json(sample.collect()[0][0])
```

## UI / product surfaces

- Connect sync for JSON-heavy sources; Catalog **schema** tab for nested columns.

## Exam traps

- **Infer schema** on inconsistent JSON → all string columns or corrupt records.
- Exploding in **bronze** instead of silver (breaks raw replay).
- Ignoring **_rescued** / malformed row handling in Auto Loader.
